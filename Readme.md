自己制作的一个运行时权限小工具，方便后续编程使用时的动态权限申请。
使用了一个button来触发运行时的权限申请，可在程序中通过注释的方式屏蔽掉自己不想要的权限
普通权限只需要在AndroidManifest中注册即可，无需动态申请。
欢迎各位朋友补充完善相关功能。
